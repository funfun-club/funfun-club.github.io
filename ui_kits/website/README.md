# UI Kit — 面白い活動 公式サイト

A hi-fi recreation of the public-facing homepage (広報ページ). Single-page site that functions as a hub pointing to YouTube and SNS.

## Structure
- `index.html` — full page, renders all sections
- `Nav.jsx` — sticky top nav with magenta rule
- `Hero.jsx` — gradient hero with logo + tagline
- `VideoGrid.jsx` — latest videos, thumbnail cards
- `SnsPanel.jsx` — giant SNS link row
- `Footer.jsx` — minimal footer with copyright and hanko stamp

## Screens
This site IS a single screen, but we expose 3 hero variations via Tweaks:
1. Gradient-wash hero (default)
2. VHS-dark hero with neon text
3. Paper hero with brush stroke title
